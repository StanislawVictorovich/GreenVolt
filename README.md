Mini ERP (Vue 2.6 + Vuex, CDN, Bootstrap 5.3)

Запуск:
- Открыть index.html (можно прямо с диска).

Логины:
- admin / admin (ADMIN) — может удалять детали/комплектующие/устройства и сбрасывать базу.
- manager / manager (MANAGER) — может добавлять и собирать.

Функции:
- Детали: добавление, закупка (поступление), удаление (ADMIN).
- Комплектующие: создание, рецепт (из деталей), сборка со списанием деталей.
- Устройства: создание, рецепт (из комплектующих), сборка со списанием комплектующих.
- Хранение: localStorage, авто-сохранение.
- Авторизация: cookie на 24 часа.
- Бэкап: экспорт/импорт JSON.
