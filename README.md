# Mini ERP (Vue 2 + Vuex + Bootstrap 5, без npm)

## Запуск
1. Распакуйте архив.
2. Откройте `index.html` в браузере.

Важно: проект использует CDN (Vue 2.6.x, Vuex, Vue Router, Bootstrap 5.3). Интернет нужен только для загрузки CDN.

## Данные
- Все данные автоматически сохраняются в `localStorage`.
- Экспорт/импорт всей базы: верхнее меню **База → Сохранить в файл / Восстановить из файла**.
- Сброс базы: **База → Сбросить базу**.

## Роли и доступ
В демо есть локальная “авторизация” (без сервера), роли ограничивают доступ к разделам.
Тестовые логины:
- `admin / admin`
- `manager / manager`

Управление пользователями: раздел **Пользователи** (доступно только Admin).

## Логика склада
Остатки считаются по движениям (ledger):
- Закупка (проведение) добавляет детали/комплектующие на склад.
- Производство (проведение) списывает вход по рецепту и добавляет выход с себестоимостью.
- Продажа (проведение) списывает изделия и считает прибыль (выручка − себестоимость).

Себестоимость — средневзвешенная (по накопленным движениям).
